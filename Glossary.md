# IT Glossary

---

## Hypervisor

Software that allows one physical computer to run multiple virtual machines.

Example:
Hyper-V is Microsoft's hypervisor.

---

## Virtual Machine (VM)

A software-based computer that has its own operating system, memory, storage, and networking while sharing the physical hardware of the host computer.

---

## Host

The physical computer that runs the hypervisor.

Example:
My Windows 11 Pro desktop is the host.

---

## Guest

The operating system running inside a virtual machine.

Example:
Ubuntu is a guest operating system.

---

## ISO

A single file that contains an exact copy of an installation disc.

Example:
The Ubuntu installation ISO is mounted to install Ubuntu in a virtual machine.

---

## External Virtual Switch

Connects virtual machines to the physical network, allowing them to communicate with other devices and access the Internet.
